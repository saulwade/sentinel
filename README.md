# Sentinel

**DevTools for AI agents.** Watch them. Rewind them. Fork reality.

Powered by Claude Opus 4.7. Built for the [Built with Opus 4.7 Hackathon](https://cerebralvalley.ai/events/~/e/built-with-4-7-hackathon).

See [`docs/`](./docs) for product design, technical plan, and the demo script.

## Quick start

```bash
pnpm install
cp apps/engine/.env.example apps/engine/.env  # add your ANTHROPIC_API_KEY
pnpm dev
```

Then open [http://localhost:3000](http://localhost:3000).
